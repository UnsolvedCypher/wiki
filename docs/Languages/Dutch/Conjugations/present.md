| | |
---|---
- | infinitive

# Completely irregular

Very few verbs are completely irregular, but the few that are are very important. Here's a list:

<div class="split " markdown="1">

<div markdown="1">

## zijn = to be

| | |
--|--
ben | zijn
bent | zijn
is | zijn

</div>

<div markdown="1">

## hebben = to have
| | |
--|--
heb | hebben
hebt | hebben
heeft | hebben

</div>

</div>

| | |
--|--

</div>
